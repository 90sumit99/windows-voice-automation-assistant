# Windows Voice Automation Assistant (Offline)

This project is an offline Windows-based voice automation assistant built using
C++ (Win32 API) and Python.

## Features (Planned - Version 1)
- Open system applications using voice or text commands
- Offline execution (no internet dependency)
- Secure command execution using whitelisting
- Modular architecture (C++ core + Python NLP)

## Tech Stack
- C++ (Win32 API)
- Python (Offline NLP & Speech)
- Windows OS
